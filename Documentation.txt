- Sieve of Eratosthenes = Fastest prime finder in range of 10^6
Syntax:
const int MAX_LIMIT = 1000000;
vector<bool> is_prime(MAX_LIMIT + 1, true);

void sieve() {

    is_prime[0] = false;
    is_prime[1] = false;

    int limit = (int) sqrt(MAX_LIMIT);

    for(int i = 2; i <= limit; i++) {
        if (is_prime[i]) {
            for(int j = i*i; j <= MAX_LIMIT; j+=i) {
                is_prime[j] = false;
            }
        }
    }
}
Explanation: It's make a vector with all the digits from 0 to 10^6 and assume every digit is prime. Then make 0 and 1 false or not prime. Then set 2 as prime and make every other digit divisible by 2 false. Then set 3 prime and make every other digit divisible by 3 false. And so on. Example:

Premade vector
[true, true, true, true, true, true, true, true, true, true, true, true, ......]

After the loop
[false, false, true, true, false, true, false, true, false, false, false, true, ......]

Now, to check if a number is prime, check that index if it's true. If so yes or else no.
